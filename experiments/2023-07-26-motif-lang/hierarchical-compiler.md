# Hierarchy

- Goal
  - Non-deterministic
  - Self-contained
- Task
  - Deterministic tools
  - Non-deterministic parameter
  - Coupled to goal and data
- Machine code
  - Fully deterministic
  - Coupled to task and data

## JIT-compile

- Goal + data => Task
- Task + data => Machine code

## Portability

Recompile task and machine code layers
